Patch 3.1 (April 11th, 2018)

NEW

  • SSH Key Tool: generate new SSH keys for all nodes.
  • Configure IP Tool: configure a static IP and unique hostname for nodes.
  • Update Tool: remotely update all packages for the nodes.
  • Nodes File: file to hold all configured nodes.
  • Base Net: wireless access point running on boot, removing need for monitor.

FIXES

  • Removed service restart from network-manager.
  • Fixed linking of wifi adaptor for networks (Mode 1).
  • All tools now use the Nodes file created by Config IP Tool.

IMPROVEMENTS

  • Tool output has been improved
  • Now ignoring non rpicluster nodes that own a DNS lease (like your computer!)
